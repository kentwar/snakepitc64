How to Run the Game:

Option 1: Direct Download

  1. Click the "Download Game APK" button above
  2. Save the APK file to your computer
  3. Install Python 3.6+ if you don't have it already
  4. Install Pygame: pip install pygame
  5. Extract the APK file (it's a ZIP file with a different extension)
  6. Run the main.py file: python main.py

Option 2: Clone from GitHub

  1. Clone the repository: git clone https://github.com/kentwar/snakepitc64.git
  2. Install dependencies: pip install pygame numpy
  3. Run the game: python -m main

The APK file contains all game assets and code in a single file.
