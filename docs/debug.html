Troubleshooting

  • If the game shows a black screen, try clicking on the canvas or refreshing the page.
  • Make sure you're using a modern browser with WebAssembly support.
  • If using Safari on iOS, make sure JavaScript is enabled.
  • Try disabling any content blockers or browser extensions.
  • Check your browser's console for error messages (F12 or right-click > Inspect > Console).

Compatible Browsers

  • Chrome 57+
  • Firefox 53+
  • Safari 11+
  • Edge 16+
  • Opera 44+
