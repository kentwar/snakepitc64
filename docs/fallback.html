Download & Play

Download the game to play locally with Python on your computer.

Download Options

Option 1: Try Another Browser

The web version works best on these browsers:

  • Google Chrome (recommended)
  • Mozilla Firefox
  • Microsoft Edge
Note: Safari and mobile browsers may have limited compatibility with the Python/Pygbag version.

Option 2: Download and Run Locally

You can download the game and run it on your computer:

  1. Make sure you have Python 3.6+ installed
  2. Clone or download the repository: https://github.com/kentwar/snakepitc64
  3. Install the required dependencies:
pip install pygame numpy
  1. Run the game:
python -m main
Note: The desktop version requires a keyboard to play.

Option 3: Diagnose Your System

If you're having trouble with the game, you can:

  • Run our diagnostic tool to check your browser compatibility
  • Check browser console for errors (F12 or right-click > Inspect > Console)
  • Report an issue on GitHub
