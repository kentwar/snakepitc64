Snakepit C64 Remake

A modern Python remake of the classic C64 game Snakepit.

Launch Game

How to Play:

Once the game launches:

  1. Click anywhere on the screen to allow the game to capture keyboard input.
  2. Use the arrow keys to control your character.
  3. Collect eggs while avoiding snakes.
  4. Press P to pause the game.
  5. Press ESC to quit.

Troubleshooting:

If you experience a black screen or the game doesn't start:

  • Make sure you're using a modern browser (Chrome, Firefox, or Edge recommended).
  • Click on the game canvas to give it focus.
  • Check your browser console for any errors (F12 key).
  • Try refreshing the page.
